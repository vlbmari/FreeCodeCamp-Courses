Grace Hopper

Rear Admiral Grace Hopper: Software Engineering and the Standardization of COBOL

  • 1906 – Born in New York City, New York.
  • 1943 – Joined the United States Naval Reserve during World War II.
  • 1944 – Assigned to the Harvard Mark I computer project, becoming one of its first programmers.
  • 1947 – Documented the first actual case of a 'bug' being found (a moth stuck in a relay) in the Mark II logbook.
  • 1952 – Developed the A-0, the first compiler, which translated mathematical code into machine-readable instructions.
  • 1959 – Led the committee that created COBOL, a high-level programming language designed to be readable and hardware-independent.
